题目链接：https://leetcode-cn.com/problems/powx-n/

题目描述：实现 pow(x, n) ，即计算 x 的 n 次幂函数。

解题思路：
    本题使用快速幂算法：尽量使得计算路径为x -> x^2 -> x^4 -> x^8 ->...。该算法可以将时间复杂度控制在O(logn)。
    对于那些并非2的整数次幂的次数计算，有如下规律：
       给定n = 95，则计算路径为1 -> 2 -> 2*2+1 -> 2*5+1 -> 2*11+1 ->2*23+1 -> 2*47+1。
       可以看出，最初的1计算了6次平方，后面引入的1分别计算了4、3、2、1次平方，即将95拆分成了64+16+8+4+2+1。
       而上述拆分的结果与95的二进制表示可以一一对应：95 = (1011111)。
